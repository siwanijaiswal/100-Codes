## [Find the Sum of N Natural Numbers](https://prepinsta.com/cpp-program/to-find-the-sum-of-n-natural-numbers/)


``` Loop ```
```cpp
#include<bits/stdc++.h>
using namespace std;

int main(){

int n;
cin>> n;
int sum=0;

for(int i=1; i<=n; i++){
    sum = sum + i;
}
cout<< " The sum of first n natural number is "<< sum <<endl;
    return 0;
}

// T.C = o(n)
// S.C = o(1)

```

``` Using Formula ```
``` cpp
#include<bits/stdc++.h>
using namespace std;

int main(){

int n;
cin>> n;

cout<< n*(n+1) / 2;

return 0;
}

//T.C = O(1)
// S.C = O(1)
